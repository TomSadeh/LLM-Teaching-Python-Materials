# Exercise Narrative Template: World Builder

**Usage:** 2% of exercises
**Best For:** OOP exercises, entity systems, simulations
**Pattern:** World Intro → Entities → Interactions → Systems → Complete World

## Key Features
- Entity creation focus
- Relationship building
- System emergence
- World simulation

## Structure
```
Opening (World introduction)
Entities: Create characters, items, locations (40%)
Interactions: Entity relationships & behaviors (40%)
Systems: Emergent systems & rules (20%)
Closing (World complete & alive)
```

## Theme Examples
- **dumblecode**: Magical kingdom building
- **chirthon**: Pantheon & realms
- **compile-games**: District & arena design
- **pyfire**: Lost Cities creation
- **pymentor**: System architecture

## Use When
- Module 9 OOP exercises
- Multiple class definitions
- Entity interaction systems
- Simulation exercises
