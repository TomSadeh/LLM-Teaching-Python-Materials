# Exercise Narrative Template: Mystery Investigation

**Usage:** 6% of exercises
**Best For:** Debug exercises, error decoding, code analysis
**Pattern:** Mystery Setup → Clue 1 → Clue 2 → Clue 3 → Solution

## Key Features
- Investigation narrative
- Evidence/clue system
- Progressive discovery
- Case resolution

## Structure
```
Opening (Mystery introduced)
Clue 1: The Error Message
Clue 2: The Variable Trail
Clue 3: The Logic Path
Clue 4: The Data Structure
Solution: Case Solved!
Closing (Case resolution)
```

## Theme Examples
- **dumblecode**: Dark magic investigation
- **chirthon**: Oracle's prophecy mystery
- **compile-games**: Arena sabotage
- **pyfire**: Ability malfunction
- **pymentor**: System debugging

## Use When
- decode_error exercises
- bug_hunt exercises
- Code analysis tasks
- Debugging focus
