# Exercise Narrative Template: Challenge Quest

**Usage:** 26% of exercises
**Best For:** Game-like progression with competitive elements
**Pattern:** Challenge Intro → Level 1 ⭐ → Level 2 ⭐⭐ → Level 3 ⭐⭐⭐ → Victory

## Key Features
- Progressive difficulty levels
- Point/score system
- Achievement tracking
- Victory conditions

## Structure
```
Opening (Challenge briefing)
Level 1: Easy (100 pts)
Level 2: Medium (200 pts)
Level 3: Hard (300 pts)
Level 4: Expert (500 pts)
Victory (Final score & achievements)
```

## Theme Examples
- **dumblecode**: House Points competition
- **chirthon**: Divine Trials
- **compile-games**: Arena rounds
- **pyfire**: Ability mastery levels
- **pymentor**: Coding challenges

## Use When
- Motivation through gamification needed
- Clear difficulty progression
- Competitive/achievement-oriented students
- Exercise types: write_code, complete_function
