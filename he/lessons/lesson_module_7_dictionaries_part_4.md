## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `.keys()` | מחזיר את כל המפתחות במילון |
| `.values()` | מחזיר את כל הערכים במילון |
| `.items()` | מחזיר את כל זוגות מפתח-ערך כטאפלים |

## תוכן השיעור

### בהמשך לחלק 3

אנחנו יודעים איך לגשת לערכים בודדים ולטפל במפתחות חסרים. עכשיו בואו נלמד לעבד את כל הרשומות במילון באמצעות לולאות.

### לולאה על מפתחות

כברירת מחדל, לולאה על מילון נותנת את המפתחות:

```python
character = {"name": "{{hero}}", "age": 11, "house": "{{house}}"}

for key in character:
    print(key)
# פלט:
# name
# age
# house
```

זה זהה לשימוש ב-`.keys()`:

```python
for key in character.keys():
    print(key)
```

### לולאה על ערכים

להשתמש ב-`.values()` לקבלת רק הערכים:

```python
character = {"name": "{{hero}}", "age": 11, "house": "{{house}}"}

for value in character.values():
    print(value)
# פלט:
# {{hero}}
# 11
# {{house}}
```

### לולאה על שניהם (זוגות מפתח-ערך)

להשתמש ב-`.items()` לקבלת גם מפתחות וגם ערכים:

```python
character = {"name": "{{hero}}", "age": 11, "house": "{{house}}"}

for key, value in character.items():
    print(f"{key}: {value}")
# פלט:
# name: {{hero}}
# age: 11
# house: {{house}}
```

### הבנת `.items()`

שיטת `.items()` מחזירה זוגות (טאפלים). התחביר `for key, value` פורק כל זוג:

```python
# זה מה ש-.items() מחזיר:
print(list(character.items()))
# [('name', '{{hero}}'), ('age', 11), ('house', '{{house}}')]
```

### לשים לב

- **ניסיון לפרוק בלולאה רגילה**: `for key, value in character:` לא יעבוד - צריך `.items()`
- **שכחה איזו שיטה להשתמש**: keys() למפתחות, values() לערכים, items() לשניהם
