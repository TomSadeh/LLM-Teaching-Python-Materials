## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `if` | מריץ את הקוד המוזח רק כשהתנאי הוא True |
| אופרטורי השוואה | `==` (שווה), `!=` (לא שווה), `<`, `>`, `<=`, `>=` |
| בוליאני (Boolean) | ערך שהוא או `True` או `False` |

---

## תוכן השיעור

### התחלה
- להתחיל עם המקרה הפשוט ביותר: `if condition:`
- להשתמש בדוגמה קונקרטית: בדיקה אם סיסמה מתאימה
- להראות מה קורה כשהתנאי True לעומת False

```python
password = input("Enter password: ")
if password == "{{password}}":
    print("Access granted!")
```

### המבנה של משפט if
1. מילת המפתח `if`
2. תנאי (משהו שהוא True או False)
3. נקודתיים `:`
4. קוד מוזח שרץ רק כשהתנאי True

### לשים לב ל
- **נקודתיים חסרות**: `if x == 5` בלי ה-`:` גורם לשגיאה
- **אין הזחה**: פייתון משתמש בהזחה כדי לדעת מה "בתוך" ה-if
- **שימוש ב-`=` במקום `==`**: `=` מקצה ערכים, `==` משווה אותם
