## תוכן השיעור

### בניה על חלק 1

למדנו איך להדפיס טקסט ישירות. עכשיו נאחסן ערכים כדי שנוכל להשתמש בהם מספר פעמים.

### חומר חדש

להציג את האנלוגיה של "קופסה עם תווית" - משתנה הוא כמו קופסה עם שם עליה:

```python
name = "Maya"
print(name)
```

נקודות מפתח:
- `name` הוא המשתנה (התווית על הקופסה)
- `"Maya"` הוא הערך (מה שבתוך הקופסה)
- `=` שם את הערך במשתנה (השמה)

להראות שכשמשתמשים במשתנה, לא משתמשים בגרשיים:
```python
name = "Maya"
print(name)    # מדפיס: Maya
print("name")  # מדפיס: name (המילה, לא הערך!)
```

משתנים יכולים לאחסן גם מספרים:
```python
age = 12
print(age)
```

### נקודות לשים לב

- **גרשיים סביב שמות משתנים**: `print("name")` מדפיס את המילה, לא את הערך
- **שגיאות הקלדה בשמות משתנים**: `print(nmae)` כשהמשתנה הוא `name`
- **רגישות לאותיות גדולות/קטנות**: `Name` ו-`name` הם משתנים שונים
