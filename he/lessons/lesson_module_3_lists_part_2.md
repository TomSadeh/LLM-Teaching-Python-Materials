## תוכן השיעור

### בהמשך לחלק 1

עכשיו שאתם יודעים ליצור רשימות ולגשת לפריטים, בואו נלמד איך לשנות אותן. רשימות הן ניתנות לשינוי (mutable), כלומר אפשר לשנות אותן אחרי היצירה.

### שינוי פריט

השתמשו באינדקס כדי להחליף פריט:

```python
fruits = ["apple", "banana", "cherry"]
fruits[1] = "blueberry"  # מחליף את "banana"
print(fruits)  # ["apple", "blueberry", "cherry"]
```

### הוספת פריטים עם append()

השיטה `append()` מוסיפה פריט לסוף:

```python
fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits)  # ["apple", "banana", "cherry"]

fruits.append("date")
print(fruits)  # ["apple", "banana", "cherry", "date"]
```

### הסרת פריטים עם remove()

השיטה `remove()` מסירה את ההתאמה הראשונה:

```python
fruits = ["apple", "banana", "cherry"]
fruits.remove("banana")
print(fruits)  # ["apple", "cherry"]
```

### לשים לב ל

- **ניסיון לגשת לפריטים שהוסרו**: אחרי הסרה, האינדקסים משתנים
- **שגיאות תחביר עם append**: זה `fruits.append("item")`, לא `fruits.append = "item"`
- **remove() על פריט חסר**: גורם לשגיאה אם הפריט לא ברשימה
