## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `math.sqrt(x)` | מחזיר את השורש הריבועי של x |
| `math.pow(x, y)` | מחזיר x בחזקת y |
| `math.floor(x)` | מעגל למטה למספר השלם הקרוב |
| `math.ceil(x)` | מעגל למעלה למספר השלם הקרוב |
| `math.pi` | הקבוע המתמטי פאי (3.14159...) |

## תוכן השיעור

### בהמשך לחלק 3

למדנו datetime - עכשיו בואו נחקור math. מודול math מספק פונקציות מתמטיות מעבר לאופרטורים הבסיסיים.

### פונקציות Math נפוצות

```python
import math

# שורש ריבועי
print(math.sqrt(16))    # 4.0
print(math.sqrt(2))     # 1.4142...

# חזקות (אלטרנטיבה ל-**)
print(math.pow(2, 3))   # 8.0

# עיגול
print(math.floor(3.7))  # 3 (תמיד מעגל למטה)
print(math.ceil(3.2))   # 4 (תמיד מעגל למעלה)
print(round(3.5))       # 4 (מובנה, מעגל לקרוב)
```

### קבועים מתמטיים

```python
import math

print(math.pi)   # 3.141592653589793
print(math.e)    # 2.718281828459045

# חישוב שטח מעגל
radius = 5
area = math.pi * radius ** 2
```

### מתי להשתמש ב-math לעומת אופרטורים מובנים

| פעולה | מובנה | מודול math |
|-------|-------|------------|
| חזקה | `2 ** 3` | `math.pow(2, 3)` |
| ערך מוחלט | `abs(-5)` | `math.fabs(-5)` |
| שורש ריבועי | `x ** 0.5` | `math.sqrt(x)` |

להשתמש במודול math עבור: בהירות, פונקציות מיוחדות (sqrt, floor, ceil), קבועים (pi, e)

### לשים לב ל

- בלבול בין `math.pow()` לאופרטור `**` (שניהם עובדים, אבל `**` לעתים קרובות פשוט יותר)
- שכחה ש-floor/ceil מחזירים מספרים שלמים קונספטואלית אבל טיפוס float בפייתון 2 (int בפייתון 3)
