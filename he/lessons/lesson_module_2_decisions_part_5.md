## תוכן השיעור

### בהמשך לחלק 4
- למדנו לשלב תנאים עם `and`, `or`, `not`
- לפעמים אנחנו צריכים החלטות שתלויות בהחלטות קודמות

### חומר חדש

#### if בתוך if אחר
```python
print("You enter a dark cave...")
choice1 = input("Go left or right? ")

if choice1 == "left":
    print("You find a treasure chest!")
    choice2 = input("Open it? (yes/no) ")
    if choice2 == "yes":
        print("Gold coins spill out!")
    else:
        print("You leave it alone.")
else:
    print("You find a friendly {{creature}}!")
```

#### מתי לקנן לעומת מתי להשתמש ב-`and`
```python
# גישת קינון
if has_ticket:
    if age >= 13:
        print("Welcome!")

# שימוש ב-and (לרוב נקי יותר)
if has_ticket and age >= 13:
    print("Welcome!")
```

**להשתמש בקינון כאשר**:
- הודעות/פעולות שונות לכל רמה
- ההחלטה הפנימית הגיונית רק אם התנאי החיצוני True

**להשתמש ב-`and` כאשר**:
- צריך רק ששני התנאים יהיו True
- יש רק תוצאה אחת שאכפת לנו ממנה
