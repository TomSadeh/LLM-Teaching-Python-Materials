## תוכן השיעור

### בהמשך לחלק 2

למדתם ליצור, לגשת, ולשנות רשימות. עכשיו בואו נלמד לעבד כל פריט ברשימה באופן אוטומטי באמצעות לולאות.

### לולאת for-in

זו הדרך הנקייה ביותר לעבור על רשימה:

```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# פלט:
# apple
# banana
# cherry
```

המשתנה `fruit` מקבל כל ערך ברשימה, אחד בכל פעם.

### כשצריך גם את האינדקס

השתמשו ב-`enumerate()` כדי לקבל גם את המיקום וגם את הפריט:

```python
fruits = ["apple", "banana", "cherry"]

for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# פלט:
# 0: apple
# 1: banana
# 2: cherry
```

### בניית רשימה חדשה מישנה

```python
numbers = [1, 2, 3, 4, 5]
doubled = []

for num in numbers:
    doubled.append(num * 2)

print(doubled)  # [2, 4, 6, 8, 10]
```

### לשים לב ל

- **שינוי הרשימה בזמן הלולאה**: זה גורם לדילוג על פריטים או שגיאות; לעבוד עם עותק במקום
- **שימוש ב-range(len(list)) שלא לצורך**: `for item in list` נקי ויותר פייתוני
