## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `random.randint(a, b)` | מחזיר מספר שלם אקראי בין a ל-b (כולל) |
| `random.choice(list)` | מחזיר פריט אקראי מרשימה |
| `random.shuffle(list)` | מסדר מחדש פריטים ברשימה באקראי (משנה את הרשימה ישירות) |

## תוכן השיעור

### בהמשך לחלק 1

עכשיו שאנחנו מבינים תחביר ייבוא, בואו נתרגל עם מודול שתלמידים כבר ראו - מודול random. זה חימום טוב לפני לימוד מודולים חדשים.

### חזרה על פונקציות Random

```python
import random

# מספר שלם אקראי בין 1 ל-10
number = random.randint(1, 10)

# בחירה אקראית מרשימה
spells = ["{{spell1}}", "{{spell2}}", "{{spell3}}"]
chosen = random.choice(spells)

# ערבוב רשימה במקום
cards = ["A", "2", "3", "4", "5"]
random.shuffle(cards)  # cards עכשיו מעורבב
```

### בדיקת ביטחון מהירה

לפני שממשיכים, לוודא שהתלמיד יכול:
- לכתוב משפט import
- להשתמש בתחביר module.function()
- להסביר מה כל פונקציית random עושה

### לשים לב ל

- שכחה ש-`shuffle()` משנה את הרשימה ישירות ומחזיר None
- בלבול בין `randint` (כולל) ל-`range` (סוף לא כולל)
