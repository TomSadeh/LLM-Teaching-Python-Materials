## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| רשימה (List) | אוסף מסודר של פריטים המאוחסנים במשתנה אחד |
| אינדקס (Index) | מיקום של פריט ברשימה, מתחיל מ-0 |
| אינדקס שלילי | ספירה מסוף הרשימה, כאשר -1 הוא הפריט האחרון |

---

## תוכן השיעור

### התחלה

רשימה היא כמו מיכל שמחזיק פריטים מרובים בסדר. במקום ליצור משתנים נפרדים לכל פריט, אפשר לאחסן את כולם במקום אחד.

```python
# יצירת רשימה עם סוגריים מרובעים
fruits = ["apple", "banana", "cherry"]
numbers = [10, 20, 30, 40, 50]
mixed = ["hello", 42, True]  # רשימות יכולות להכיל טיפוסים שונים
```

### גישה לפריטים לפי אינדקס

לכל פריט יש מספר מיקום שנקרא אינדקס. פייתון מתחילה לספור מ-0, לא מ-1.

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # "apple" - פריט ראשון
print(fruits[1])  # "banana" - פריט שני
print(fruits[2])  # "cherry" - פריט שלישי
```

### אינדקס שלילי

השתמשו במספרים שליליים כדי לספור מהסוף:

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[-1])  # "cherry" - פריט אחרון
print(fruits[-2])  # "banana" - שני מהסוף
```

### לשים לב ל

- **טעויות של אחד**: זכרו שאינדקס 0 הוא הפריט הראשון, לא אינדקס 1
- **אינדקס מחוץ לטווח**: גישה ל-`fruits[3]` כשברשימה יש רק 3 פריטים (אינדקסים 0, 1, 2)
