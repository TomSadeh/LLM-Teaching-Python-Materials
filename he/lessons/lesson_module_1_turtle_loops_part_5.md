## תוכן השיעור

### בהמשך לחלק 4

עכשיו שאנחנו יודעים לצייר צורות, בואו נעשה אותן צבעוניות ומעניינות יותר!

### שינוי צבעים

```python
t.color("red")      # עכשיו הצב מצייר באדום
t.forward(100)

t.color("blue")     # מחליפים לכחול
t.forward(100)
```

צבעים נפוצים: "red", "blue", "green", "yellow", "purple", "orange", "pink"

### שינוי מהירות

```python
t.speed(1)   # איטי - טוב לצפייה במה שקורה
t.speed(5)   # בינוני
t.speed(10)  # מהיר
t.speed(0)   # הכי מהיר (בלי אנימציה)
```

### צבעים בלולאה

אפשר להשתמש ברשימת צבעים ולעבור עליהם בלולאה:

```python
colors = ["red", "orange", "yellow", "green", "blue", "purple"]

for color in colors:
    t.color(color)
    t.forward(100)
    t.right(60)
```

### לולאות מקוננות לדפוסים מורכבים

רוצים לצייר כמה ריבועים? שמים לולאה בתוך לולאה:

```python
for i in range(6):           # תעשה את זה 6 פעמים
    for j in range(4):       # צייר ריבוע
        t.forward(100)
        t.right(90)
    t.right(60)              # פנה לפני הריבוע הבא
```

זה מצייר 6 ריבועים, כל אחד מסובב 60 מעלות!

### לשים לב

- שמות צבעים חייבים להיות בגרשיים ("red" ולא red)
- לשכוח באיזו רמת לולאה נמצאים עם לולאות מקוננות
- הזחה הופכת לקריטית עם לולאות מקוננות
