## תוכן השיעור

### בניה על חלק 1

עכשיו שאנחנו יודעים איך לעשות לולאה "עד שמשהו קורה", אנחנו צריכים משהו בלתי צפוי שיקרה. כאן נכנסים מספרים אקראיים - הם הופכים משחקים למרגשים כי אי אפשר לנחש את התוצאה!

### חומר חדש

קודם כל, צריך לייבא את המודול random:
```python
import random
```

אז אפשר לייצר מספרים אקראיים:
```python
import random

number = random.randint(1, 10)
print(number)  # יכול להיות 1, 2, 3, 4, 5, 6, 7, 8, 9, או 10
```

נקודה מרכזית: `randint` כולל את שני הקצוות! זה שונה מ-`range()`.

להריץ את אותו קוד כמה פעמים - תקבלו תוצאות שונות בכל פעם!

### דוגמת קוביות

```python
import random

die1 = random.randint(1, 6)
die2 = random.randint(1, 6)
print(f"הטלת: {die1} ו-{die2}")
print(f"סכום: {die1 + die2}")
```

### לשים לב ל

- **שכחת `import random`**: חייב להיות בראש הקובץ
- **תחביר לא נכון**: זה `random.randint()`, לא סתם `randint()`
- **חשיבה של אחד בצד**: `randint(1, 10)` כן כולל 10

## אם התלמיד מתקשה

- **אם לא מקבל אקראיות**: להסביר ש-`randint` באמת אקראי - אי אפשר לנחש אותו
- **אם מבולבל לגבי range לעומת randint**: `range(1, 10)` לא כולל 10, אבל `randint(1, 10)` כן כולל
- **אם שוכח את הייבוא**: הודעת השגיאה תגיד "name 'random' is not defined" - זה אומר שצריך לייבא
