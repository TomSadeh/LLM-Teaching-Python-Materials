## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `__init__` | שיטה מיוחדת שרצה אוטומטית כשיוצרים אובייקט חדש |
| `self` | הפניה לאובייקט הנוכחי שנוצר או בשימוש |
| תכונה (Attribute) | משתנה ששייך לאובייקט (למשל, `self.name`) |

## תוכן השיעור

### בניה על חלק 2

יצרנו אובייקטים והוספנו תכונות אחר כך. אבל יש דרך טובה יותר - להגדיר תכונות בזמן היצירה.

### הבעיה עם הוספת תכונות מאוחר יותר

```python
hero = Character()
hero.name = "{{hero}}"
hero.health = 100

# מה אם שוכחים אחת?
sidekick = Character()
sidekick.name = "{{friend}}"
# אופס! שכחנו להגדיר health!
print(sidekick.health)  # שגיאה!
```

### הפתרון: __init__

```python
class Character:
    def __init__(self, name, health):
        self.name = name
        self.health = health

# עכשיו חייבים לספק name ו-health כשיוצרים
hero = Character("{{hero}}", 100)
sidekick = Character("{{friend}}", 80)
```

### הבנת self

`self` מתייחס לאובייקט הספציפי שנוצר:

```python
def __init__(self, name, health):
    self.name = name    # שמירת name על האובייקט הזה
    self.health = health  # שמירת health על האובייקט הזה
```

כשקוראים ל-`Character("{{hero}}", 100)`:
- פייתון יוצר אובייקט חדש
- `self` מתייחס לאובייקט החדש הזה
- `self.name = name` שומר "{{hero}}" על האובייקט

### מופעים שונים, ערכים שונים

```python
hero = Character("{{hero}}", 100)
villain = Character("{{villain}}", 120)

print(hero.name)     # {{hero}}
print(villain.name)  # {{villain}}
```

כל אובייקט שומר את הערכים שלו!

### לשים לב ל

- **אי שימוש ב-self.**: כתיבת רק `name = name` יוצרת משתנה מקומי, לא תכונה
- **שכחת פרמטר self**: `def __init__(name, health):` לא יעבוד
- **בלבול בין פרמטר לתכונה**: הפרמטר `name` והתכונה `self.name` הם דברים שונים

## נקודת בדיקה

לשאול: "בשביל מה `__init__`?"
צפוי: היא רצה כשיוצרים אובייקט חדש ומגדירה את הערכים ההתחלתיים שלו

---

## אם התלמיד מתקשה

- **אם __init__ מסתורי**: לקרוא לזה "פונקציית ההגדרה" שרצה אוטומטית כשיוצרים אובייקט
- **אם self מבלבל**: להסביר את זה כ"האובייקט הזה כאן" - כמו לומר את השם שלך כשאתה מתייחס לעצמך
- **אם לא שומר עם self.**: לצייר דיאגרמה שמראה את ההבדל בין משתנה מקומי `name` לתכונת אובייקט `self.name`
