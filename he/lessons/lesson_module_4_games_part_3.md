## תוכן השיעור

### בניה על חלקים 1-2

אנחנו יודעים לולאות while (חזרה עד שמשהו נכון) ומספרים אקראיים (ערכים בלתי צפויים). עכשיו נשלב אותם כדי ליצור את המשחק הראשון שלנו!

### חומר חדש

מבנה משחק ניחוש המספר הקלאסי:
```python
import random

secret = random.randint(1, 100)
guess = 0  # אתחול למשהו שהוא לא הסוד

while guess != secret:
    guess = int(input("נחש את המספר: "))

    if guess < secret:
        print("נמוך מדי!")
    elif guess > secret:
        print("גבוה מדי!")
    else:
        print("מצאת!")
```

אלמנטים מרכזיים:
1. המחשב בוחר מספר אקראי פעם אחת (לפני הלולאה)
2. הלולאה ממשיכה עד שהשחקן מנחש נכון
3. לתת רמזים אחרי כל ניחוש לא נכון
4. לקבל ניחוש חדש בכל מעבר בלולאה

### לשים לב ל

- **אי עדכון הניחוש בתוך הלולאה**: הניחוש חייב להשתנות, אחרת זו לולאה אינסופית
- **בחירת מספר אקראי חדש בכל איטרציה**: הסוד צריך להיקבע לפני הלולאה, לא בתוכה
- **אי המרת קלט ל-int**: `input()` מחזיר מחרוזת, צריך `int()` להשוואת מספרים

## אם התלמיד מתקשה

- **אם הלוגיקה של המשחק לא נכונה**: לפרק לחתיכות קטנות יותר - קודם להפעיל את המספר האקראי, ואז להוסיף את הלולאה
- **אם הלולאה אף פעם לא נגמרת**: לבדוק ש-`guess` מתעדכן בתוך הלולאה
- **אם המספר האקראי ממשיך להשתנות**: לוודא ש-`random.randint()` נקרא פעם אחת, לפני לולאת ה-while
