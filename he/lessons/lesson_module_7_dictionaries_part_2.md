## תוכן השיעור

### בהמשך לחלק 1

בחלק 1, למדנו ליצור מילונים ולקרוא ערכים. עכשיו נלמד לשנות אותם - להוסיף נתונים חדשים, לעדכן ערכים קיימים, ולהסיר רשומות.

### הוספת זוגות מפתח-ערך חדשים

פשוט להקצות למפתח שעדיין לא קיים:

```python
character = {"name": "{{hero}}", "age": 11}
character["house"] = "{{house}}"
print(character)
# פלט: {'name': '{{hero}}', 'age': 11, 'house': '{{house}}'}
```

### שינוי ערכים קיימים

להקצות ערך חדש למפתח קיים:

```python
character = {"name": "{{hero}}", "age": 11}
character["age"] = 12  # יום הולדת!
print(character["age"])  # פלט: 12
```

### הסרת רשומות

להשתמש ב-`del` להסרת זוג מפתח-ערך:

```python
character = {"name": "{{hero}}", "age": 11, "temporary_item": "map"}
del character["temporary_item"]
print(character)
# פלט: {'name': '{{hero}}', 'age': 11}
```

### לשים לב

- **ניסיון להשתמש ב-append()**: תלמידים עשויים לנסות `character.append()` - זה לרשימות! במילונים, פשוט להקצות למפתח חדש
- **בלבול בין שינוי להוספה**: התחביר זהה - Python יוצר מפתח חדש אם הוא לא קיים, או מעדכן אם הוא קיים

## אם התלמיד מתקשה

- **אם מנסה להשתמש ב-append()**: להסביר ש-append() הוא לרשימות (הוספה לסוף). למילונים אין "סוף" - יש להם מפתחות. להראות: `dict["new_key"] = value`
- **אם מבולבל לגבי הוספה לעומת שינוי**: להצביע שפייתון מטפל בשני המקרים אוטומטית. אם המפתח קיים, הוא מעדכן. אם לא, הוא יוצר.
