## תוכן השיעור

### בהמשך לחלק 1
- בחלק 1 כתבנו קוד שרץ רק כשתנאי הוא True
- אבל מה אם אנחנו רוצים שמשהו יקרה כשהוא False?

### חומר חדש
- להציג את מושג ה"אחרת" עם `else`
- להראות if-else כשני מסלולים: אחד או השני רץ, אף פעם לא שניהם

```python
password = input("Enter password: ")
if password == "{{password}}":
    print("Access granted!")
else:
    print("Access denied!")
```

### המבנה
1. משפט ה-`if` עם התנאי והנקודתיים שלו
2. קוד מוזח למקרה True
3. מילת המפתח `else` ואחריה נקודתיים (בלי תנאי!)
4. קוד מוזח למקרה False

### לשים לב ל
- **הזחה של else ברמה לא נכונה**: `else` חייב להיות מיושר עם ה-`if` שלו
- **הוספת תנאי ל-else**: `else` לא מקבל תנאי - הוא תופס את כל מה שלא היה True
- **שכחת הנקודתיים אחרי else**: בדיוק כמו `if`, גם `else` צריך `:`

## אם התלמיד מתקשה

- **אם מבולבל לגבי מתי else רץ**: לצייר שני מסלולים - "אם הסיסמה נכונה, לך שמאלה; אחרת, לך ימינה"
- **אם מקבל שגיאות הזחה**: לוודא ש-else מיושר עם if, ולשניהם יש בלוקים מוזחים משלהם
