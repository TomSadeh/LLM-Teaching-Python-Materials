# מודול 0: בסיס פייתון - חימום!

## מה המודול הזה מכסה
תרגול של מה שכבר את יודעת לפני שממשיכים!

- `print()` - להציג פלט
- משתנים - `str` ו-`int`
- `input()` - לקבל קלט מהמשתמש
- מתמטיקה בסיסית
- לולאות `for` עם `range()`
- פונקציות - `def`, פרמטרים, `return`
- מודול `turtle` - ציור בסיסי עם הצב
