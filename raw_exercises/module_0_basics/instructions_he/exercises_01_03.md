# תרגילים 1-3: Print, משתנים, מתמטיקה

## תרגיל 1: Hello World ו-Print
פשוט הדפיסי דברים!

```python
print("Hello!")        # מדפיס Hello!
print("שורה 1")
print("שורה 2")        # כל print בשורה חדשה
```

## תרגיל 2: משתנים
משתנים שומרים מידע:

```python
name = "Maya"          # מחרוזת (str)
age = 12               # מספר שלם (int)
print(name)            # מדפיס: Maya
print("Hi " + name)    # מדפיס: Hi Maya
```

**שימי לב:**
- שמות משתנים רגישים לאותיות גדולות/קטנות
- `myName` ו-`myname` הם משתנים שונים!

## תרגיל 3: מתמטיקה
פייתון יכול לחשב:

```python
print(5 + 3)    # 8 (חיבור)
print(10 - 4)   # 6 (חיסור)
print(6 * 7)    # 42 (כפל)
print(20 / 4)   # 5.0 (חילוק)
print(2 ** 3)   # 8 (חזקה: 2 בחזקת 3)
print(17 % 5)   # 2 (שארית: 17 חלקי 5 = 3 שארית 2)
```

**סדר פעולות חשבון:**
- סוגריים קודם
- אחר כך חזקות
- אחר כך כפל/חילוק
- אחר כך חיבור/חיסור
