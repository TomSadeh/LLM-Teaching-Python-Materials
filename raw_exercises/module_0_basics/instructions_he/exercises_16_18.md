# תרגילים 16-18: היכרות עם מודול הצב (Turtle)

## מה זה Turtle?
מודול `turtle` מאפשר לך לצייר על המסך! יש לך "צב" שאפשר להזיז, והוא משאיר קו אחריו.

## פקודות בסיסיות

```python
import turtle           # טוענים את המודול

t = turtle.Turtle()     # יוצרים צב חדש

t.forward(100)          # לך קדימה 100 פיקסלים
t.backward(50)          # לך אחורה 50 פיקסלים
t.right(90)             # פנה ימינה 90 מעלות
t.left(45)              # פנה שמאלה 45 מעלות

turtle.done()           # שומר את החלון פתוח
```

## תרגיל 17: ציור צורות (בלי לולאות!)

### חלק א׳ - ריבוע
ציירי ריבוע על ידי כתיבת כל צעד בנפרד.

```
קדימה 100, ימינה 90
קדימה 100, ימינה 90
קדימה 100, ימינה 90
קדימה 100, ימינה 90
```

**שאלה:** רואה כמה פעמים את חוזרת על אותו דבר? בשיעור הבא נלמד לולאות שיעשו את זה קצר יותר!

### חלק ב׳ - משולש
ציירי משולש.

**רמז חשוב:** במשולש, כל פנייה היא **120 מעלות** (לא 90!)
- למה? כי 360 ÷ 3 = 120

### חלק ג׳ - בית
ציירי בית פשוט: ריבוע עם גג משולש.

**טיפ:** אחרי שמציירים את הריבוע, הצב חוזר לנקודת ההתחלה. צריך להזיז אותו למעלה לפני שמציירים את הגג.

## תרגיל 18: צבעים ושליטה בעט

### פקודות חדשות

```python
t.color("red")          # שנה צבע העט
t.penup()               # הרם את העט (לא מצייר)
t.pendown()             # הורד את העט (מתחיל לצייר)
t.speed(3)              # מהירות (1=איטי, 10=מהיר)
```

### צבעים זמינים
`"red"`, `"blue"`, `"green"`, `"purple"`, `"orange"`, `"pink"`, `"yellow"`, `"brown"`, `"black"`

### חלק א׳
ציירי קו אדום, פני ימינה, ציירי קו כחול.

### חלק ב׳
ציירי שני קווים **נפרדים** (עם רווח ביניהם).

**רמז:** השתמשי ב-`penup()` כדי לזוז בלי לצייר, ואז `pendown()` כדי להתחיל לצייר שוב.

### חלק ג׳
ציירי דפוס צבעוני - 4 קווים בצבעים שונים!
