##  转
- 专 转转 专  注砖转
- 砖转 注专 (砖, , 拽)
- 砖转砖 -`if`, `elif` -`else`
- 爪专 转转 专拽转

## 砖注专

### 砖注专 1: 砖驻 if
转转 转 拽 转!  注  砖专 转 转!

```python
house = "Gryffindor"
if house == "Gryffindor":
    print("Where dwell the brave at heart!")
```

### 砖注专 2: 驻专专 砖
砖转砖   砖转 注专:
- `==` 砖 (砖 住 砖!)
- `!=`  砖
- `>`  
- `<` 拽 
- `>=`   砖
- `<=` 拽  砖

```python
district = 12
if district == 12:
    print("You're from the coal mining district, like Katniss!")
```

### 砖注专 3: if/else
  转  ? 砖转砖 -`else`:

```python
# 专转 砖 拽转 转 住住!
password = input("Password? ")
if password == "Caput Draconis":
    print("The portrait swings open. Welcome to Gryffindor!")
else:
    print("The Fat Lady shakes her head.")
```

### 砖注专 4: elif (else if)
专 驻砖专转, 砖转砖 -`elif`:

```python
# 爪 O.W.L. 专住!
score = 85
if score >= 95:
    print("O - Outstanding!")
elif score >= 85:
    print("E - Exceeds Expectations!")
elif score >= 70:
    print("A - Acceptable")
else:
    print("T - Troll!")
```
