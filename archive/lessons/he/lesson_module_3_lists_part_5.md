### בהמשך לחלקים 1-4

עכשיו יש לכם את כל אבני הבניין: יצירת רשימות, גישה לפריטים, שינוי רשימות, לולאות, ובדיקת תכנים. בואו נשלב את המיומנויות האלה לתוכניות אמיתיות.

### בחירה אקראית מרשימות

```python
import random

responses = ["כן!", "בשום פנים", "אולי", "שאל שוב מאוחר יותר"]
answer = random.choice(responses)
print(answer)
```

### שילוב רשימות עם תנאים

```python
scores = [85, 92, 78, 95, 88]
high_scores = []

for score in scores:
    if score >= 90:
        high_scores.append(score)

print(f"ציונים גבוהים: {high_scores}")  # [92, 95]
```

### בניית אפליקציית רשימה אינטראקטיבית

```python
playlist = []

# הוספת שירים
playlist.append("שיר א")
playlist.append("שיר ב")
playlist.append("שיר ג")

# הצגת הכל
print("הפלייליסט שלך:")
for i, song in enumerate(playlist):
    print(f"{i + 1}. {song}")

# בדיקה והסרה
if "שיר ב" in playlist:
    playlist.remove("שיר ב")
    print("הוסר שיר ב")
```

### לשים לב ל

- **שכחה לייבא random**: חייב להיות `import random` בראש
- **שגיאות רשימה ריקה**: `random.choice([])` על רשימה ריקה גורם לשגיאה
