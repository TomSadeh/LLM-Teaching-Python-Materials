## תוכן השיעור

### בהמשך לחלק 3

עכשיו שאתם יכולים לעבור על רשימות בלולאה, בואו נלמד פעולות חזקות לבדיקה וחילוץ מידע מרשימות.

### קבלת אורך עם len()

```python
fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # 3

# שימושי ללולאות ותנאים
if len(fruits) > 0:
    print("הרשימה לא ריקה")
```

### בדיקת חברות עם in

אופרטור `in` מחזיר True או False:

```python
fruits = ["apple", "banana", "cherry"]

print("banana" in fruits)   # True
print("mango" in fruits)    # False

# שימוש נפוץ עם משפטי if
if "apple" in fruits:
    print("יש לנו תפוחים!")
```

### חיתוך רשימות

קבלו חלק מרשימה עם `[start:end]`:

```python
numbers = [0, 1, 2, 3, 4, 5]

print(numbers[1:4])   # [1, 2, 3] - מאינדקס 1 עד (לא כולל) 4
print(numbers[:3])    # [0, 1, 2] - מההתחלה עד אינדקס 3
print(numbers[3:])    # [3, 4, 5] - מאינדקס 3 עד הסוף
print(numbers[-2:])   # [4, 5] - 2 פריטים אחרונים
```

### לשים לב ל

- **בלבול len() עם האינדקס האחרון**: אם `len(fruits)` הוא 3, האינדקס האחרון הוא 2
- **סוף החיתוך לא כולל**: `[1:4]` נותן אינדקסים 1, 2, 3 (לא 4)
