## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `print()` | מציג טקסט על המסך לבני אדם לראות - הערך "נצרך" |
| `return` | שולח ערך חזרה לקוד שקרא לפונקציה - אפשר להשתמש בערך |

## תוכן השיעור

### בנייה על חלק 3

זו ההבחנה הכי חשובה בשיעור הזה! הרבה מתחילים מבלבלים בין `print` ל-`return`. בואו נבהיר את זה לחלוטין.

### השוואה זה לצד זה

```python
# גרסה 1: שימוש ב-print
def double_print(number):
    result = number * 2
    print(result)

# גרסה 2: שימוש ב-return
def double_return(number):
    result = number * 2
    return result
```

הם נראים דומים, אבל מתנהגים מאוד אחרת!

### ההבדל הקריטי

```python
# עם print - לא יכולים להשתמש בתוצאה
answer1 = double_print(5)
print(f"answer1 is: {answer1}")  # answer1 is: None

# עם return - יכולים להשתמש בתוצאה
answer2 = double_return(5)
print(f"answer2 is: {answer2}")  # answer2 is: 10
```

`double_print(5)` מציג "10" על המסך, אבל `answer1` הוא `None`.
`double_return(5)` נותן חזרה 10, אז `answer2` הוא `10`.

### למה זה חשוב

לנסות להשתמש בפונקציה שרק מדפיסה בחישוב:

```python
def add_print(a, b):
    print(a + b)

def add_return(a, b):
    return a + b

# זה נכשל!
total = add_print(2, 3) + add_print(4, 5)
# מדפיס 5, אחר כך 9, אחר כך קורס: TypeError: unsupported operand type(s) for +: 'NoneType' and 'NoneType'

# זה עובד!
total = add_return(2, 3) + add_return(4, 5)
print(total)  # 14
```

### מתי להשתמש במה

| להשתמש ב-`print()` כש... | להשתמש ב-`return` כש... |
|--------------------------|-------------------------|
| רוצים להציג משהו למשתמש | צריכים להשתמש בתוצאה בקוד |
| לדיבוג (לראות מה קורה) | לחישובים ועיבוד נתונים |
| לפלט סופי בסוף תוכנית | כשקוד אחר צריך את הערך |

### דוגמה מעשית: מאמתים

```python
def is_valid_password(password):
    if len(password) >= 8:
        return True    # Return, לא print!
    else:
        return False

# עכשיו אפשר להשתמש בתנאים
user_password = input("Enter password: ")
if is_valid_password(user_password):
    print("Password accepted!")
else:
    print("Password too short!")
```

אם היינו משתמשים ב-`print(True)` במקום `return True`, משפט ה-`if` לא היה עובד!

## אם התלמיד מתקשה

- **אם הבלבול print/return נמשך**: לצייר תרשים שמראה לאן ערכים "הולכים" עם כל גישה
  - Print: הערך הולך למסך (מוצג ונעלם)
  - Return: הערך חוזר לקוד הקורא (אפשר לשמור ולהשתמש)
- **אם עדיין מבולבל**: להשתמש באנלוגיה פיזית - print זה כמו לצעוק תשובה בקול (כולם שומעים, אבל היא נעלמה), return זה כמו למסור למישהו פתק (הם יכולים לשמור ולהשתמש)
- **אם לא רואה את ההבדל המעשי**: לבקש מהם לכתוב תוכנית שצריכה לבדוק מספר תנאים באמצעות פונקציות שרק מדפיסות - הם יראו שזה לא עובד
