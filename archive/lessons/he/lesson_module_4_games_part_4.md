## תוכן השיעור

### בניה על חלק 3

משחק הניחוש שלנו עובד, אבל אין ניקוד! בואו נוסיף מעקב מצב כדי להפוך אותו ליותר מרתק.

### חומר חדש

הוספת מונה ניסיונות:
```python
import random

secret = random.randint(1, 10)
attempts = 0  # לעקוב אחר כמה ניחושים

while True:
    guess = int(input("נחש (1-10): "))
    attempts = attempts + 1

    if guess == secret:
        print(f"נכון! לקח לך {attempts} ניסיונות.")
        break
    elif guess < secret:
        print("נמוך מדי!")
    else:
        print("גבוה מדי!")
```

הגבלת ניחושים (הוספת קושי):
```python
import random

secret = random.randint(1, 10)
max_attempts = 3
attempts = 0

while attempts < max_attempts:
    guess = int(input("נחש: "))
    attempts = attempts + 1

    if guess == secret:
        print("ניצחת!")
        break
    else:
        remaining = max_attempts - attempts
        print(f"לא נכון! נשארו {remaining} ניחושים.")

if guess != secret:
    print(f"סוף המשחק! המספר היה {secret}")
```

### לשים לב ל

- **איפוס ניקוד בתוך הלולאה**: משתני מצב צריכים להיות מאותחלים לפני הלולאה
- **שכחה להגדיל**: `attempts = attempts + 1` חייב להיות בתוך הלולאה
- **טעות של אחד בגבול**: להשתמש ב-`<` לעומת `<=` נכון למספר הנכון של ניסיונות

## אם התלמיד מתקשה

- **אם הניקוד ממשיך להתאפס**: לבדוק ש-`score = 0` הוא לפני לולאת ה-while, לא בתוכה
