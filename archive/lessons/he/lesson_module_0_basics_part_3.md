## תוכן השיעור

### בניה על חלק 2

אנחנו יכולים לאחסן מספרים במשתנים. עכשיו בואו נעשה חשבון איתם!

### חומר חדש

להראות פעולות חשבון בסיסיות:
```python
print(2 + 3)   # חיבור: 5
print(10 - 4)  # חיסור: 6
print(3 * 5)   # כפל: 15
print(10 / 2)  # חילוק: 5.0
```

לשלב עם משתנים:
```python
age = 12
next_year = age + 1
print(next_year)  # 13
```

אפשר לעשות חשבון ישירות ב-print:
```python
print(5 + 3)  # 8
```

### נקודות לשים לב

- **ניסיון לעשות חשבון על מחרוזות**: `"5" + 3` גורם לשגיאה
- **שימוש ב-`x` לכפל**: פייתון משתמש ב-`*`, לא ב-`x`
- **חילוק תמיד נותן עשרוניים**: `10 / 2` נותן `5.0`, לא `5`
