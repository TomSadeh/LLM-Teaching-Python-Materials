## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `return` | שולח ערך חזרה מהפונקציה למקום שבו היא נקראה |
| ערך מוחזר (Return value) | הערך שפונקציה נותנת חזרה כשהיא מסיימת |

## תוכן השיעור

### בנייה על חלק 2

למדנו לשלוח מידע לתוך פונקציה באמצעות פרמטרים. עכשיו נלמד לקבל מידע בחזרה מפונקציה באמצעות `return`.

### הצורך בערכי החזרה

לפעמים אנחנו רוצים שפונקציה תחשב משהו ותיתן לנו את התוצאה:

```python
# הפונקציה הזו מחשבת, אבל אנחנו לא יכולים להשתמש בתוצאה!
def add_numbers(a, b):
    result = a + b
    print(result)  # רק מציג, לא יכולים להשתמש במקום אחר

add_numbers(5, 3)  # מדפיס 8, אבל אי אפשר לעשות איתו כלום
```

### הצגת Return

```python
def add_numbers(a, b):
    result = a + b
    return result  # לשלוח את התוצאה בחזרה!

# עכשיו אפשר לתפוס ולהשתמש בתוצאה
total = add_numbers(5, 3)
print(f"The total is {total}")  # The total is 8
print(total * 2)                # 16 - אפשר להשתמש!
```

### דוגמה מעשית: המרת טמפרטורה

```python
def celsius_to_fahrenheit(celsius):
    fahrenheit = (celsius * 9/5) + 32
    return fahrenheit

# שימוש בערך המוחזר
temp_f = celsius_to_fahrenheit(25)
print(f"25C is {temp_f}F")  # 25C is 77.0F

# אפשר להשתמש גם בתנאים
if celsius_to_fahrenheit(30) > 80:
    print("It's hot!")
```

### מה קורה בלי Return?

```python
def no_return_example():
    x = 5 + 3
    # אין משפט return

result = no_return_example()
print(result)  # None - שום דבר לא הוחזר!
```

אם לפונקציה אין משפט `return`, היא מחזירה `None` כברירת מחדל.

### Return מסיים את הפונקציה

קוד אחרי `return` לא רץ:

```python
def example():
    return 42
    print("This never runs!")  # השורה הזו לעולם לא מגיעה

value = example()  # value is 42
```

### לשים לב ל

- **שכחת return**: לחשב ערך אבל לא להחזיר אותו
- **קוד אחרי return**: לשים קוד אחרי משפט ה-return (הוא לא ירוץ)
- **לא לתפוס את הערך המוחזר**: לקרוא לפונקציה אבל לא לשמור את התוצאה

## אם התלמיד מתקשה

- **אם שוכח לתפוס ערך מוחזר**: להראות זה לצד זה - `add(2, 3)` לעומת `result = add(2, 3)`
- **אם מבולבל לאן הערך הולך**: לעקוב דרך הקוד צעד אחר צעד, להראות את הערך "נוסע" חזרה
- **אם שם קוד אחרי return**: להסביר ש-return זה כמו דלת יציאה - ברגע שיצאת, אי אפשר לעשות עוד בחדר הזה
