## תוכן השיעור

### בניה על חלק 4

אנחנו יכולים לקבל קלט ולעשות חשבון. עכשיו בואו נשלב הכל לתוכניות שלמות!

### חומר חדש

**שיטה 1: שרשור (חיבור עם +)**
```python
name = "Maya"
print("Hello, " + name + "!")
```

**שיטה 2: f-strings (מומלץ)**
```python
name = "Maya"
age = 12
print(f"Hello, {name}! You are {age} years old.")
```

f-strings נקיים יותר - פשוט לשים `f` לפני הגרשיים ולהשתמש ב-`{}` סביב משתנים.

**דוגמה מלאה:**
```python
name = input("What is your name? ")
age = int(input("How old are you? "))
next_year = age + 1

print(f"Hello, {name}!")
print(f"Next year you will be {next_year}.")
```

### נקודות לשים לב

- **שכחת ה-`f`**: `"{name}"` לא עובד בלי הקידומת `f`
- **חסרות סוגריים מסולסלות**: `f"Hello, name"` מדפיס "name" ממש
- **ערבוב סוגי שרשור**: אי אפשר לעשות `"Hi " + 5` (צריך `str(5)` או להשתמש ב-f-string)
