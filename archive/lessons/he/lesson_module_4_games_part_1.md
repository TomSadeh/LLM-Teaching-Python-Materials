## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| לולאת `while` | חוזרת כל עוד תנאי הוא True (בניגוד ל-`for` שחוזר מספר קבוע של פעמים) |
| תנאי לולאה | הביטוי הבוליאני שקובע אם הלולאה ממשיכה או עוצרת |
| לולאה אינסופית | לולאה שאף פעם לא עוצרת כי התנאי תמיד True |

---

## תוכן השיעור

### התחלה

להשוות בין `while` ל-`for`:
- לולאת `for`: "תעשה את זה בדיוק N פעמים"
- לולאת `while`: "תעשה את זה כל עוד משהו נכון"

דוגמת ספירה לאחור פשוטה:
```python
count = 5
while count > 0:
    print(count)
    count = count - 1
print("שיגור!")
```

נקודה מרכזית: התנאי נבדק *לפני* כל איטרציה. אם הוא מתחיל כ-False, גוף הלולאה אף פעם לא ירוץ.

### דוגמה: להמשיך לשאול

תבנית נפוצה היא לבקש קלט עד שהוא תקין:
```python
answer = ""
while answer != "כן":
    answer = input("האם אתה מוכן? ")
print("יאללה!")
```

### לשים לב ל

- **לולאות אינסופיות**: שכחה לעדכן את המשתנה שבתנאי
- **טעויות של אחד**: הלולאה רצה פעם אחת יותר מדי או פחות מדי
- **אי כניסה ללולאה**: התנאי הוא False מההתחלה
