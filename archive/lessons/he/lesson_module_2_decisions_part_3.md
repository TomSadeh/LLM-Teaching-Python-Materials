## תוכן השיעור

### בהמשך לחלק 2
- עם if-else, יש לנו שני מסלולים: True או False
- מה אם אנחנו צריכים יותר משתי אפשרויות?

### חומר חדש
- לבנות מערכת ציונים: A, B, C, D, F
- להראות איך `elif` בודק בסדר ועוצר בהתאמה הראשונה

```python
score = int(input("Enter score: "))
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")
```

### למה הסדר חשוב
- `elif` בודק תנאים בסדר
- הוא עוצר בתנאי הראשון שהוא True
- אם למישהו יש 95: גם `>= 90` וגם `>= 80` הם True, אבל אנחנו רוצים רק "A"

### מספר if לעומת elif
```python
# לא נכון: מספר if (כולם נבדקים)
if score >= 90:
    print("A")
if score >= 80:  # זה גם רץ עבור ציון 95!
    print("B")

# נכון: elif (עוצר בהתאמה הראשונה)
if score >= 90:
    print("A")
elif score >= 80:  # זה לא רץ עבור ציון 95
    print("B")
```

### לשים לב ל
- **סדר תנאים לא נכון**: בדיקת `>= 60` לפני `>= 90` תיתן לכולם D
- **שימוש במספר if במקום elif**: כל משפטי ה-`if` נבדקים; `elif` עוצר בהתאמה הראשונה
