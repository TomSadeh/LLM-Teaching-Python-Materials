---

## מושגי מפתח

| מושג | הסבר במשפט אחד |
|------|----------------|
| `break` | יוצא מיד מהלולאה הנוכחית |
| לולאת תפריט | לולאת while True שמציגה אפשרויות ויוצאת כשהמשתמש בוחר לצאת |
| `while True` | תבנית נפוצה ללולאות שיוצאות דרך break ולא דרך תנאי |

---

## תוכן השיעור

### בניה על חלק 4

לפעמים אנחנו רוצים לצאת מלולאה מיד, לא לחכות עד שהתנאי ייבדק. זה מה ש-`break` עושה.

### חומר חדש

שימוש ב-`break` ליציאה:
```python
while True:
    choice = input("לשחק שוב? (כן/יציאה): ")
    if choice == "יציאה":
        print("תודה ששיחקת!")
        break
    print("מתחיל משחק חדש...")
```

משחק מונע תפריט:
```python
while True:
    print("\n=== תפריט משחק ===")
    print("1. שחק משחק")
    print("2. צפה בשיא")
    print("3. יציאה")

    choice = input("בחר: ")

    if choice == "1":
        print("משחק...")
        # קוד המשחק כאן
    elif choice == "2":
        print("שיא: 100")
    elif choice == "3":
        print("להתראות!")
        break
    else:
        print("בחירה לא תקינה, נסה שוב.")
```

נקודה מרכזית: `break` יוצא רק מהלולאה הפנימית ביותר. אם יש לולאות מקוננות, break יוצא רק מהפנימית.

### לשים לב ל

- **שימוש ב-break מחוץ ללולאה**: `break` חייב להיות בתוך לולאת while או for
- **בלבול break ו-return**: `break` יוצא מהלולאה; `return` יוצא מהפונקציה
- **בלבול break ו-continue**: `break` יוצא לגמרי; `continue` מדלג לאיטרציה הבאה

## אם התלמיד מתקשה

- **אם break לא עובד**: לוודא שהוא בתוך הלולאה, לא מחוצה לה
- **אם מבולבל לגבי break לעומת continue**: "break = לעזוב את הבניין; continue = לדלג על החדר הזה, ללכת לבא"
- **אם בלבול עם לולאות מקוננות**: רק הלולאה הפנימית ביותר מושפעת מ-break
