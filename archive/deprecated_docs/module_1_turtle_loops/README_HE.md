# מודול 1: אומנות צב עם לולאות

## מה תלמדי
- שימוש בלולאות `for` כדי לחזור על פעולות
- לגרום לצב לצייר צורות
- ליצור דפוסים מגניבים עם קוד

## שיעורים

### שיעור 1: צורות בסיסיות
לפני שנעשה אומנות, בואי נתרגל לצייר צורות עם לולאות!

**תזכורת - ככה נראית לולאת for:**
```python
for i in range(4):
    print("זה מודפס 4 פעמים!")
```

הצב מבין פקודות כמו:
- `t.forward(100)` - זוז קדימה 100 פיקסלים
- `t.right(90)` - פנה ימינה 90 מעלות
- `t.left(45)` - פנה שמאלה 45 מעלות
- `t.penup()` - הפסק לצייר
- `t.pendown()` - התחל לצייר שוב
- `t.color("red")` - שנה את צבע העט

### שיעור 2: משתנים שולטים בהכל
את יכולה להשתמש במשתנים כדי לשנות את הציורים שלך:
```python
size = 100
for i in range(4):
    t.forward(size)
    t.right(90)
```

נסי לשנות את `size` - מה קורה?

### שיעור 3: לולאות מקוננות (לולאות בתוך לולאות!)
רוצה לצייר צורה כמה פעמים? שימי לולאה בתוך לולאה!
